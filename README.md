# 👩🏻‍🔬 Testing with Jest
This repo contains some beginner-friendly examples of tested functions .

## Installation
``` bash
npm i
```
## Usage
In order to run the tests you might run this command:
```bash
npm test
```
If you want to run one test file only, you would prefer:

```bash
npm test {foo.test.js}
```
